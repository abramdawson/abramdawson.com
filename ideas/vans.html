Abram / Ideas: Vans

Drivers
  1. Wifi + 5G enable Internet anywhere. Internet enables productivity. "Can work from anywhere, expected to work from everywhere."
  2. Home ownership feels too far away. Want to start building equity without settling on one specific location.
  3. Jobs don't mean careers. Higher turnover, people staying at one place for less time, want ability to move around while still building equity.
  4. Technology, better batteries, solar. With autonomous not too far away, idea of self-driving house considerably more compelling.
  5. The wealthy solve this by owning multiple homes, can travel between places and live for periods of time. Airbnb proves demand.
Thoughts
  1. Cities need to repurpose old infrastructure. Parking garages.
  2. Each "unit" requires ammenities: bathroom/shower, electricity, water, bike locker, outdoor space. Prices based on amenities (top floor/natural light).
  3. Scale using tech. Bathrooms are self cleaning, entirely power-washable - nozzels embedded in walls/ceilings/floors + UV light.
  4. Community is important since preseumably close quarters. Need to build right behavior, create accountability. Value of area goes down with bad people (Burning Man).
